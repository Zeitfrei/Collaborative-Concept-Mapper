Bachelorarbeit zur Entwicklung eines Moodle Plug-Ins für kollaboratives Concept Mapping.

Aufbauend auf einem bestehendem Concept Mapping Tool wurde im Rahmen der Arbeit mit Hilfer der JavaScript Bibliothek TogetherJS ein Plug-In für Moodle entwickelt, welches es Studenten erlaubt gemeinsam Concept Maps zu erstellen.
